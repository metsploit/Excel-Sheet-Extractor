# 📊 Excel Sheet Extractor

A Python-powered desktop application that instantly splits large Excel master workbooks into individual, perfectly formatted standalone workbooks in seconds.

Designed for analysts, operations teams, reporting professionals, and anyone who regularly works with large Excel files.

---

## 🚀 The TL;DR

Instead of spending hours manually copying worksheets into new files, this tool lets you:

✅ Select a master workbook

✅ Choose the sheets you want

✅ Rename output files if required

✅ Generate separate Excel workbooks while preserving formatting

All in just a few clicks.

---

## ✨ Features

- 🎛️ Clean Desktop GUI built with Tkinter
- 📂 Browse any Excel workbook (.xlsx / .xlsm)
- ✅ Select or Deselect all worksheets instantly
- ✏️ Rename output files before extraction
- 🎨 Preserves formatting including:
  - Fonts
  - Colors
  - Borders
  - Cell alignment
  - Row heights
  - Column widths
  - Merged cells
  - Hidden rows & columns
- ⚡ Batch extraction of dozens of worksheets in seconds
- 💻 No command line knowledge required

---

# 🛑 The Problem

If you've ever worked in Operations, Supply Chain, Finance, E-commerce, or Reporting, you've probably dealt with a massive Excel workbook containing multiple worksheets.

Examples include:

- Regional Reports
- Inventory Reports
- Vendor Reports
- Marketplace Reports
- Forecast Files
- Daily Operations Dashboards

When different teams only need specific worksheets, Excel forces you to repeat the same process over and over.

1. Right-click worksheet
2. Move or Copy
3. Create New Workbook
4. Save As
5. Rename file
6. Repeat...

Do this 20–50 times every day, and you've lost hours of productivity.

---

# 💡 The Solution

Excel Sheet Extractor automates this entire workflow.

Simply:

1. Open your master workbook
2. Select the worksheets
3. Rename them if needed
4. Click Extract

The tool automatically creates separate Excel files for every selected worksheet while keeping the formatting intact.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/metsploit/Excel-Sheet-Extractor.git
```

Go inside the project

```bash
cd excel-sheet-extractor
```

Install dependencies

```bash
pip install openpyxl
```

Run the application

```bash
python "SHEET EXTRACTER 2.py"
```
